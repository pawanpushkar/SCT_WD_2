# ⏱ Advanced Stopwatch Web Application

## 📌 Project Overview

This project is part of the **SkillCraft Technology Internship – Task 02**.

The objective was to build an **interactive and user-friendly stopwatch web application** with advanced features like lap tracking and precise time measurement.

---

## 🚀 Features

* ✅ Start, Pause, and Reset functionality
* ✅ Millisecond precision timing
* ✅ Lap time recording with numbering
* ✅ Real-time display (HH:MM:SS:MS)
* ✅ Responsive and modern UI (Glassmorphism design)
* ✅ Smooth animations and transitions

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and modern UI effects
* **JavaScript** – Timer logic and interactivity

---

## ⚙️ How It Works

* The stopwatch uses `setInterval()` for time tracking
* Time is updated every **10 milliseconds**
* Values are formatted using `padStart()` for consistency
* Lap times are dynamically added to the DOM
* Button states are controlled for better user experience
